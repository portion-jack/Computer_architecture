# 5. CPU 성능 향상 기법

## CPU의 언어
ISA(Intruction Set Architecure)
    - 1. CISC(Complex Instruction Set Computer)
        - 가변 길이 명령어
        - 메모리를 절약 가능하지만,
        - 실행시간이 일정하지 않아서 명령어 파이프라인 구현시, 시간 딜레이가 필수적이다.
     
    - 2. RISC(Reduced Instruction Set Computer)
        - 고정 길이 명령어
        - 명령어 파이프라인 구현시 매우 편리하다.
